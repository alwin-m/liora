**Liora: Kitten Edition** is a special, "cute" version of the Liora menstrual cycle tracking application. While maintaining the core privacy-first and on-device data storage principles of the original Liora, this edition is designed for younger users and cat lovers who want a more playful, funny, and engaging experience.

---

## ✨ Kitten Edition Features

- 🐾 **Purr-fect UI** - A new, cute aesthetic with pastel colors, kitten icons, and playful animations.
- 🐈 **Cat-themed Personalization** - Custom icons and logos featuring our feline friends.
- 🎀 **Youth-Centric Design** - simplified and more engaging interface for younger audiences.
- 🔒 **Privacy First** - Exactly like the original, all health data stays 100% local on your phone.
- 🔮 **Smart Predictions** - The same powerful AI-driven cycle predictions, delivered with a touch of cuteness.

---

## 🎯 Our Mission & Goals

The Liora project is dedicated to:
1. **Research & Learning** - Exploring Flutter-based application development and cycle tracking algorithms.
2. **Technological Innovation** - Building robust, local-first applications using modern tech stacks like Flutter and Netlify.
3. **Privacy Advocacy** - Proving that health tracking can be functional without compromising user data to the cloud.
4. **Empowerment** - Providing tools for individuals to understand and manage their reproductive health.
